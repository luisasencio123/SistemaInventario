# Sistema de Gestión de Inventarios Mejorado

## Descripción
Este proyecto es un sistema de gestión de inventarios desarrollado en Python.
Permite almacenar productos en un archivo de texto y recuperar la información automáticamente al iniciar el programa.

## Funcionalidades
- Añadir productos
- Actualizar productos
- Eliminar productos
- Mostrar productos
- Guardar información en archivo inventario.txt
- Cargar inventario automáticamente al iniciar
- Manejo de excepciones (FileNotFoundError, PermissionError, ValueError)

## Tecnologías utilizadas
- Python 3
- Manejo de archivos
- Manejo de excepciones

## Cómo ejecutar el programa
1. Abrir la carpeta del proyecto.
2. Ejecutar el comando:
   python inventario.py
